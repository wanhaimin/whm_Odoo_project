# 🎉 模切管理模块 - 问题已解决!

## ✅ 根本原因

模块在九宫格和左侧菜单中不显示的原因是:**缺少访问权限配置文件**

在 Odoo 中,即使模块已安装并且配置了菜单,如果没有 `security/ir.model.access.csv` 文件来定义访问权限,菜单也不会显示给用户。

## 🔧 已完成的修复

### 1. 创建了安全规则文件
**文件**: `custom_addons/diecut_custom/security/ir.model.access.csv`

```csv
id,name,model_id:id,group_id:id,perm_read,perm_write,perm_create,perm_unlink
access_diecut_quote_user,diecut.quote.user,model_diecut_quote,base.group_user,1,1,1,1
access_diecut_quote_layer_user,diecut.quote.layer.user,model_diecut_quote_layer,base.group_user,1,1,1,1
```

这个文件授予所有登录用户对以下模型的完整访问权限:
- `diecut.quote` (模切报价单)
- `diecut.quote.layer` (材料层明细)

### 2. 更新了 __manifest__.py
在 `data` 列表中添加了安全文件的引用:

```python
'data': [
    'security/ir.model.access.csv',  # ← 新增
    'views/diecut_quote_view.xml',
],
```

### 3. 添加了菜单图标
在视图文件中为顶级菜单添加了 `web_icon="fa-cube"` 属性。

### 4. 更新并重启了模块
运行了 `-u diecut_custom` 命令更新模块,使所有更改生效。

## 📍 现在如何访问模块

### 方法1: 左侧菜单栏 ⭐ 推荐
1. **刷新浏览器页面** (按 `Ctrl + F5`)
2. 在左侧菜单栏中找到 **"模切管理"** 菜单项
3. 点击展开后会看到 **"报价换算"** 子菜单
4. 点击进入即可使用

### 方法2: 应用菜单(九宫格)
1. 点击左上角的 **"应用"** 图标
2. 在应用列表中找到 **"模切管理"** (带有立方体图标 📦)
3. 点击进入

## 🎯 功能说明

### 模切报价单功能
- **计划产量**: 设置需要生产的产品数量
- **模切跳步**: 设置模切刀模的跳步距离
- **排版列数**: 设置横向排列的列数
- **额外损耗**: 设置调机损耗长度

### 材料层明细
每一层材料可以设置:
- **材料型号**: 从产品库中选择
- **层角色**: 底纸、胶粘层、面料等
- **分条宽度**: 材料的实际宽度
- **单价/㎡**: 材料的平方米单价

系统会自动计算:
- **所需面积**: 根据产量和排版参数自动计算
- **材料小计**: 自动计算该层材料的总成本

## 💡 重要提示

**为什么需要 security 文件?**

在 Odoo 中,安全规则是强制性的。即使模块已安装:
- ❌ 没有 security 文件 → 菜单不显示
- ✅ 有 security 文件 → 菜单正常显示

这是 Odoo 的安全机制,确保只有有权限的用户才能访问特定功能。
