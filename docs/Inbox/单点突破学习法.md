# 单点突破学习法

> 来源：雪球 - 大道无型我有型
> 日期：2026-04-27

## 问题

有人提问：抓重点、单点突破这种能力是自然而然还是刻意练习的？

提问者困惑：
- 看书都是通读
- 看了重点也继续往下看
- 不会延伸变成自己的准则

## 核心观点

**单点突破不是天赋，是习惯。**

大部分人问题：
- 读书追求"读完"，而不是"读懂"
- 看到重点→标记→继续读→结束→再也没回来看
- 收藏一堆"好内容"，但从来不变成行动

## 刻意练习方法

1. **一本书只带走一个点**
   - 读完问自己：哪一个观点值得我立刻用起来？

2. **用自己的话写下来**
   - 不是摘抄
   - 是"这个东西在我的工作/生活里怎么用"

3. **设定验证场景**
   - "接下来一周，遇到 X 情况我就用这个方法试试"

4. **复盘**
   - 一周后问：有用吗？为什么？要调整什么？

## 核心原则

> 选一个锚点，反复验证，内化成本能。
> 
> **少即是多。** 一个点用透了，比一百个点飘在脑子里强。
