# 微流控热驱动阀结构示意图解读

![](Attachments/WeChat/7e549331-7734-4191-bfa0-1c198660e9eb.png)

## 图示内容
这张图展示了一种微流控 / 柔性器件中的**热驱动阀（thermal actuated valve）**结构与工作状态。

图中包含四个部分：
- **a. Top imaging**：顶视图
- **b. Side imaging**：侧视图
- **c. Section A-A, valve OFF**：A-A 截面，阀门关闭
- **d. Section A-A, valve ON**：A-A 截面，阀门开启

## 主要结构
图例中标出的关键部件包括：
- **Fluidic channel**：流体通道
- **Expansion medium**：膨胀介质
- **PDMS support layer**：PDMS 支撑层
- **PDMS fluidic layer**：PDMS 流体层
- **PDMS membrane**：PDMS 膜层
- **Epoxy glue**：环氧胶
- **Resistance wire**：电阻丝
- **Camera**：相机

## 工作原理
该结构的核心逻辑是：

1. 向 **U 形电阻丝**施加直流电（DC）
2. 电阻丝发热，产生局部温升
3. 下方的**膨胀介质**受热后体积变化
4. 推动上方 **PDMS 膜**发生形变和隆起
5. 阀门状态由 **OFF** 切换为 **ON**

也就是说，这是一种通过**热致膨胀 + 柔性膜变形**实现开关控制的微阀结构。

## 图中想表达的重点
- 从顶视图和侧视图可以看到整体布线和观察方式
- 从 A-A 截面可以看出阀门在关闭与开启状态下膜层的形变差异
- 相机用于记录阀门形变或工作过程中的成像结果

## 简要理解
这类结构常见于：
- 微流控芯片
- 柔性器件
- 实验室芯片（Lab-on-a-chip）
- 热驱动微机械系统

它的优势通常是：
- 结构相对直观
- 易于做局部驱动
- 适合小尺寸流体控制
