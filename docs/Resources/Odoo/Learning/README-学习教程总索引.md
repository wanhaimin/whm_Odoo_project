## 🎯 学习路线图

### 第一阶段:初级入门 (2 个月)

**📖 文档:** [Odoo学习教程-第一阶段-初级入门.md](./Odoo学习教程-第一阶段-初级入门.md)

**学习目标:**
- ✅ 理解 Odoo 的基本概念
- ✅ 掌握 XML 视图定义
- ✅ 学会 Domain 过滤
- ✅ 实现视图继承

**学习内容:**

| 周次 | 主题 | 重点内容 |
|------|------|----------|
| 第1周 | 认识 Odoo | Odoo 是什么、模块概念、文件结构 |
| 第2周 | XML 视图 | 表单视图、列表视图、视图结构 |
| 第3周 | Domain 过滤 | 基本语法、组合条件、动态过滤 |
| 第4周 | 视图继承 | XPath 定位、添加字段、修改属性 |
| 第5-8周 | 综合练习 | 客户管理系统项目 |

**适合人群:**
- 零基础学习者
- 想要了解 Odoo 的人
- 需要定制 Odoo 界面的用户

| 第13-16周 | 工作流 | 状态管理、按钮动作、业务流程 |

**适合人群:**
- 完成第一阶段的学习者
- 需要开发业务逻辑的开发者

### 如何使用这套教程?

#### 1. 按顺序学习
```
不要跳过章节!
每个知识点都是基础
```

#### 2. 每天学习计划

**推荐时间安排:**
- **工作日:** 每天 1-2 小时
- **周末:** 每天 2-3 小时
- **复习:** 每周末复习本周内容

**学习步骤:**
1. 阅读理论部分 (30 分钟)
2. 动手实践 (60 分钟)
3. 做练习题 (30 分钟)
4. 记笔记 (10 分钟)

**✅ 推荐做法:**
- 准备一个笔记本,记录重点
- 每个例子都要亲自动手做一遍

- 每周末复习本周内容
- 和其他学习者交流

**❌ 避免做法:**

- 跳过练习
- 遇到困难就放弃
