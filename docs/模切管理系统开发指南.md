Odoo 的列表视图（List/Tree View）对齐方式设置非常灵活，主要通过在 XML 中添加 CSS 类来实现。以下是详细的设置教程：

### 1. 核心对齐类 (CSS Classes)

Odoo 使用标准的 Bootstrap 风格类名来控制对齐。在 

 属性即可：

| 对齐方式     | 类名 (推荐用新版)                          | 旧版写法 (兼容)                         | 适用场景          |
| -------- | ----------------------------------- | --------------------------------- | ------------- |
| **左对齐**  | ```<br>class="text-start"<br>```    | ```<br>class="text-left"<br>```   | 文本、名称、描述      |
| **居中对齐** | `<br>class="text-center"%%  %%<br>` | ```<br>class="text-center"<br>``` | 状态、短代码、布尔值、图片 |
| **右对齐**  | ```<br>class="text-end"<br>```      | ```<br>class="text-right"<br>```  | 金额、数量、数字      |

### 2. 如何设置 (代码示例)



**注意：** Odoo 的设计逻辑是 **“列对齐”**。也就是说，当您设置了字段的对齐方式，**表头 (Header) 会自动跟随内容的对齐方式**。

### 3. 表头对齐的特殊说明

- **默认行为**：表头的对齐方式默认与该列的内容对齐方式一致。
    - 内容居中 -> 表头居中
    - 内容右对齐 -> 表头右对齐
- **能否“表头居中，内容居左”？**
    - **标准 XML 不支持**这种直接配置。这是 Odoo 为了保持视觉整洁做的限制。
    - 如果必须实现“表头居中，内容居左”，通常需要写自定义 CSS 或 JS，或者牺牲一下让内容也居中（对于短文本通常可以接受）。
