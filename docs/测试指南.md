## 📊 费用计算验证

使用上面的例子,费用应该是:

```
面积 = 150mm × 70mm / 100 = 105 cm²

材料成本 = 25元/m² × 105cm² × 10片 / 10000 = 2.63元
加工费用 = 105cm² × 10片 × 0.05元/cm² = 52.50元
打样费 = 100元 (正常)
总费用 = 2.63 + 52.50 + 100 = 155.13元
```

**检查:** 订单中的总费用是否接近 155元?

## ⚠️ 如果遇到问题

### 问题1: 看不到"材料管理"菜单

**解决方案:**
1. 强制刷新浏览器 (Ctrl + F5)
2. 清除浏览器缓存
3. 退出登录再重新登录
4. 检查用户是否有权限

### 问题2: 创建材料时报错

**可能原因:**
- 材料编号重复
- 必填字段未填写

**解决方案:**
- 使用唯一的材料编号
- 确保填写所有必填字段(带*号)

### 问题3: 费用计算为0

**可能原因:**
- 材料的参考价格未填写

**解决方案:**
- 编辑材料,填写参考价格
