## 🚀 下一步操作

### 1. 升级模块

运行升级脚本:
```bash
.\Upgrade_Module.bat
```

或手动升级:
```bash
python odoo-bin -c odoo.conf -d odoo_dev_new -u diecut_custom
```

### 2. 重启 Odoo

```bash
.\Restart_Odoo_Clean.bat
```

### 3. 初始化数据

#### 3.1 创建材料分类

进入 Odoo 后台:
1. 材料管理 → 材料分类
2. 创建主分类,例如:
   - 塑料类
     - PET
     - PVC
     - PC
   - 金属类
     - 铝箔
     - 铜箔
   - 胶带类
     - 双面胶
     - 泡棉胶

#### 3.2 添加材料

1. 材料管理 → 材料
2. 点击"新建"
3. 填写材料信息:
   - 基本信息(编号、名称、分类)
   - 物理特性(厚度、宽度等)
   - 性能参数
   - 商务信息(价格、供应商)
   - 上传图片和文档
4. 点击"发布到网站"

#### 3.3 测试网站功能

访问以下网址:
- 材料库: `http://localhost:8069/materials`
- 在线打样: `http://localhost:8069/sample/order`
- 我的订单: `http://localhost:8069/my/sample/orders`

## 📊 数据模型说明

### 材料分类 (material.category)
```python
- name: 分类名称
- parent_id: 父分类
- child_ids: 子分类
- material_count: 材料数量
```

### 材料 (material.material)
```python
- code: 材料编号
- name: 材料名称
- category_id: 材料分类
- manufacturer_id: 生产商
- thickness: 厚度
- reference_price: 参考价格
- website_published: 是否发布到网站
```

### 打样订单 (sample.order)
```python
- name: 订单号(自动生成)
- partner_id: 客户
- product_name: 产品名称
- line_ids: 订单明细
- quantity: 打样数量
- urgency: 紧急程度
- total_cost: 总费用(自动计算)
- state: 订单状态
```

## 💰 费用计算逻辑

### 打样费用 = 材料成本 + 加工费用 + 打样费 + 运费

#### 1. 材料成本
```
材料成本 = 材料单价(元/m²) × 面积(cm²) × 数量 / 10000
```

#### 2. 加工费用
```
加工费用 = 面积(cm²) × 数量 × 加工单价

加工单价:
- 模切: 0.05元/cm²
- 贴合: 0.08元/cm²
- 印刷: 0.10元/cm²
- 分切: 0.03元/cm²
```

#### 3. 打样费
```
基础打样费 = 100元

根据紧急程度:
- 正常(3-5天): 100元 × 1.0 = 100元
- 加急(1-2天): 100元 × 1.5 = 150元
- 特急(24小时): 100元 × 2.0 = 200元
```

## 🐛 常见问题

### Q1: 升级模块后看不到新菜单?
**A:** 清除浏览器缓存,或使用无痕模式访问

### Q2: 网站页面显示404?
**A:** 确保已安装 `website` 模块,并重启 Odoo

### Q3: 图片上传后不显示?
**A:** 检查图片格式(支持 JPG, PNG),大小不超过5MB

### Q4: 费用计算不正确?
**A:** 检查材料的参考价格是否已填写

### Q5: 客户无法下单?
**A:** 确保客户已登录,并且有门户用户权限
