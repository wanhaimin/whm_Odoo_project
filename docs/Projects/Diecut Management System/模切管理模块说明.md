# 模切管理模块 - 菜单显示说明

## ✅ 已完成的修复

### 1. 添加了 web_icon 属性
在 `diecut_quote_view.xml` 中为顶级菜单添加了 `web_icon="fa-cube"` 属性,使模块能够在 Odoo 应用菜单中显示。

### 2. 更新了模块
运行了 `odoo-bin -u diecut_custom` 命令更新模块,使更改生效。

## 📍 如何查看模块

现在您应该能在以下位置看到"模切管理"模块:

### 方法1: 左侧菜单栏
1. 刷新浏览器页面 (Ctrl + F5)
2. 在左侧菜单栏中应该能看到"模切管理"菜单项
3. 点击后会展开显示"报价换算"子菜单

### 方法2: 应用菜单(九宫格)
1. 点击左上角的"应用"图标
2. 在应用列表中应该能看到"模切管理"应用
3. 图标显示为一个立方体 (📦)

## 🔧 如果还是看不到

如果刷新后仍然看不到,请尝试:

1. **清除浏览器缓存**
   - 按 Ctrl + Shift + Delete
   - 清除缓存和 Cookie
   - 重新登录

2. **重新登录 Odoo**
   - 退出当前账户
   - 重新登录

3. **检查用户权限**
   - 确保当前用户有访问该模块的权限
   - 使用管理员账户登录

## 🎨 可用的 Font Awesome 图标

如果您想更换图标,可以修改 `web_icon` 属性为以下任一值:
- `fa-cube` - 立方体 (当前使用)
- `fa-cut` - 剪刀
- `fa-industry` - 工厂
- `fa-cogs` - 齿轮
- `fa-calculator` - 计算器
- `fa-file-text-o` - 文档

修改方法:
在 `diecut_quote_view.xml` 第39行,将 `web_icon="fa-cube"` 改为您想要的图标即可。

## 🔄 更新模块的命令

如果将来需要更新模块,使用以下命令:

```bash
.\venv\Scripts\python.exe odoo\odoo-bin -c odoo.conf -u diecut_custom --stop-after-init
```

然后重启 Odoo 服务即可。
