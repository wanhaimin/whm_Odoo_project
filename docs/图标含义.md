# 图标含义

根据您提供的图片（通常是 VS Code 的大纲视图/Structure View）以及代码文件 
diecut_quote.py
 的内容，这些图标代表了代码中的不同元素。

红框内的图标含义如下：

蓝色方括号图标 [ ] (Field / Property)
含义: 代表 字段 (Fields) 或 属性。
在您的代码中: 对应于 profit_rate, profit_amount, final_unit_price 等。
解释: 这些是您在模型中定义的变量，用于存储数据（例如 fields.Float, fields.Monetary）。
紫色立方体图标 🧊 (Method / Function)
含义: 代表 方 法 (Methods) 或 函数。
在您的代码中: 对应于 
_compute_total_material_cost
, 
create
 等。
解释: 这些是您定义的 Python 函数，用于执行逻辑（例如计算字段值、创建记录等）。
黄色节点/结构图标 (Class)
含义: 代表 类 (Classes)。
在您的代码中: 对应于底部的 
DiecutQuoteMaterialLine
 和 
DiecutQuoteManufacturingLine
。
解释: 这些是您定义的其他 Odoo 模型类。
总结： 这只是编辑器用来帮助您快速区分代码结构的视觉辅助：

蓝色 = 数据 (Data/Variables)
紫色 = 逻辑 (Logic/Functions)
黄色 = 结构 (Structure/Classes)
