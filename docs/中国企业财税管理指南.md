## 一、系统初始化教程

### 1.1 已安装模块清单

| 模块 | 功能 | 状态 |
|------|------|------|
| `account` | 核心会计引擎 | ✅ 已安装 |
| `l10n_cn` | 中国科目表、增值税税率 | ✅ 已安装 |
| `l10n_cn_tax` | 中国财税管理（多公司）| ✅ 已内置 |
| `base_accounting_kit` | 完整会计 UI（总账/试算表/现金流） | ✅ 已安装 |

安装 `base_accounting_kit` 后，顶部导航栏会出现完整的「**会计**」菜单，包含凭证、银行对账、报告等完整功能。

> 如仅显示「发票」而没有「会计」菜单，需前往 **设置 → 应用** 安装 `base_accounting_kit（会计套件）`，或联系管理员确认。

### 1.2 新建公司并选择科目表

**路径**：设置 → 公司 → 新建公司

| 字段 | 填写说明 |
|------|---------|
| 公司名称 | 营业执照全称 |
| 国家 | 中国 |
| 货币 | CNY（人民币） |
| 纳税人识别号 | 统一社会信用代码（18位） |

**选择科目表**：会计 → 配置 → 会计设置 → 财务会计（科目表）

| 公司性质 | 选择模板 |
|---------|---------|
| 小规模纳税人 | **小企业会计准则**（template\_cn） |
| 一般纳税人 | **企业会计准则**（template\_cn\_large\_bis） |

选择后系统自动创建全套科目（1xxx–5xxx）和增值税税率，**各公司数据完全隔离**。

### 1.3 配置公司税务信息

**路径**：设置 → 公司 → 编辑 → 标签页「**中国税务配置**」

| 字段 | 选项 | 说明 |
|------|------|------|
| 增值税纳税人类型 | 小规模纳税人 / 一般纳税人 | 影响增值税申报向导的计算逻辑 |
| 城建税税率 (%) | 7% / 5% / 1% | 城区 7%，县城/镇 5%，其他地区 1% |
| 小型微利企业 | 勾选/不勾选 | 影响企业所得税优惠税率判断 |

### 1.4 配置银行账户

**路径**：会计 → 配置 → 银行账户 → 新建

填写开户行名称、账号，系统自动生成对应日记账。

## 二、日常记账操作

### 2.1 客户销售开票

**路径**：会计 → 客户 → 发票 → 新建

1. 选择客户
2. 填写商品/服务行，选择税率：
   - 小规模纳税人：选「**增值税3%（小规模）**」或「**增值税1%（小规模优惠）**」
   - 一般纳税人：选 **13%**（标准货物）/ **9%**（农产品/运输/不动产）/ **6%**（服务/无形资产）
3. 点击「确认」→ 自动生成分录（借：应收账款；贷：主营业务收入 + 应交税费）

### 2.2 供应商采购入账

**路径**：会计 → 供应商 → 账单 → 新建

填写供应商、金额、税率后确认。一般纳税人收到增值税专用发票后录入进项税，系统自动抵扣。

### 2.3 费用报销

**路径**：会计 → 会计 → 凭证 → 新建

常见费用凭证结构：
```
借：管理费用 / 销售费用 / 制造费用    金额
  贷：银行存款 / 应付账款             金额
```

### 2.4 工资发放

工资计提：
```
借：管理费用—工资       应发合计
  贷：应付职工薪酬—工资    应发合计
```

工资发放：
```
借：应付职工薪酬—工资      应发合计
  贷：应交税费—应交个人所得税    代扣个税
      其他应付款—社保（个人部分）   个人社保
      银行存款                   实发工资
```

### 2.5 固定资产折旧

**路径**：会计 → 会计 → 固定资产

每月计提折旧（以设备原值 100,000 元、使用 10 年为例）：
```
借：管理费用—折旧费       833.33
  贷：累计折旧（1602）      833.33
```

> 折旧计提后，资产负债表中「固定资产净值」= 1601（原值）- 1602（累计折旧），Odoo 报表自动计算。

### 2.6 银行对账

**路径**：会计 → 银行和现金 → 对账

导入银行流水 → 系统自动匹配凭证 → 未匹配项手动核销或补录。

## 三、财务报表生成（资产负债表/利润表）

本系统已内置符合**《小企业会计准则》**格式的资产负债表和利润表，可直接从账务数据生成并打印 PDF。

### 3.1 资产负债表

**路径**：会计 → 报告 → 中国税务申报 → **资产负债表**

**操作步骤**：

1. 选择「公司」和「会计年度」
2. 选择「报表期间」（第一/二/三/四季度），系统自动设置报表基准日为该季末
3. 点击「**计算**」按钮，系统读取截止基准日的所有已过账凭证余额
4. 查看各项目金额（支持直接在页面核对）
5. 点击「**打印 PDF**」生成符合官方格式的资产负债表

**报表包含项目**：

| 资产方 | 负债及所有者权益方 |
|--------|-----------------|
| 货币资金（1001/1002/1012） | 短期借款（2001） |
| 短期投资（1101） | 应付账款（2202） |
| 应收票据（1121） | 预收账款（2203） |
| 应收账款（1122） | 应付职工薪酬（2211） |
| 预付账款（1123） | 应交税费（2221） |
| 存货（1401~1421） | 长期借款（2501） |
| 固定资产净值（1601-1602） | 实收资本（3001） |
| 无形资产净值（1701-1702） | 资本公积（3002） |
| ... | 未分配利润（3103+3104） |
| **资产总计** | **负债和所有者权益合计** |

> **注意**：固定资产净值已自动扣除累计折旧（科目 1602），无形资产净值已扣除累计摊销（科目 1702）。

**数据核对要点**：资产总计 = 负债合计 + 所有者权益合计（试算平衡）。如两者不相等，需检查未过账凭证或科目归属是否正确。

### 3.2 利润表

**路径**：会计 → 报告 → 中国税务申报 → **利润表**

**操作步骤**：

1. 选择「公司」和「会计年度」
2. 选择「报表期间」，系统自动设置日期范围为**年初（1月1日）至该季末**（即年度累计数）
3. 点击「**计算**」
4. 查看利润表各项目
5. 点击「**打印 PDF**」生成符合官方格式的利润表

**报表结构**：

```
一、营业收入（5001 主营业务收入 + 5051 其他业务收入）
  减：营业成本（5401 + 5402）
      营业税金及附加（5403）
      销售费用（5601）
      管理费用（5602）
      财务费用（5603）
  加：投资收益（5111）
二、营业利润
  加：营业外收入（5301）
  减：营业外支出（5711）
三、利润总额
  减：所得税费用（5801）
四、净利润
```

> 亏损项目在 PDF 报表中以括号显示，如 `(12,500.00)` 表示亏损 12,500 元。

### 3.3 报表生成频率建议

| 时间 | 建议动作 |
|------|---------|
| **月末** | 生成利润表（月度累计），核对营业利润是否合理 |
| **季末** | 生成资产负债表 + 利润表，配合税务申报使用 |
| **年末（12月31日）** | 生成全年资产负债表和利润表，作为汇算清缴附件存档 |

### 3.4 与 base_accounting_kit 报表的区别

| 功能 | l10n\_cn\_tax 报表（本模块） | base_accounting_kit 报表 |
|------|--------------------------|------------------------|
| 格式 | 符合《**小企业会计准则**》官方格式 | 通用国际格式 |
| 科目映射 | 按中国准则科目代码（1xxx–5xxx） | 按 account\_type 归类 |
| 打印 PDF | ✅ 支持，含中文表头和格式 | ✅ 支持，英文/通用格式 |
| 适用场景 | **对外报送、税务申报附件** | **内部管理、多维度分析** |

**建议**：税务申报和银行贷款用本模块的报表；日常管理分析用 base\_accounting\_kit 的报表。

## 四、税务申报操作（增值税/企所税）

### 4.1 增值税申报汇总

**路径**：会计 → 报告 → 中国税务申报 → **增值税申报汇总**

**操作步骤**：

1. 选择公司（多公司可切换）
2. 选择申报年份和季度
3. 点击「**计算**」

| 字段 | 说明 |
|------|------|
| 不含税销售额 | 当期已过账销售发票的不含税金额合计 |
| 销项增值税 | 销售发票税额合计 |
| 可抵扣进项税额 | **仅一般纳税人**：采购发票税额合计 |
| 实际应纳增值税 | 销项 - 进项 |
| 城建税 | 实际应纳增值税 × 城建税率（公司配置，默认7%） |
| 教育费附加 | 实际应纳增值税 × 3% |
| 地方教育费附加 | 实际应纳增值税 × 2% |
| **本期合计应缴税款** | 增值税 + 附加税三项合计 |

4. 将数据**手动填入**电子税务局增值税申报表
5. 缴税后录入纳税凭证：

```
借：应交税费—应交增值税        实际应纳增值税
    应交税费—应交城市维护建设税  城建税额
    应交税费—应交教育费附加      教育附加税额
    应交税费—应交地方教育附加    地方教育附加额
  贷：银行存款                  合计税款
```

### 4.2 企业所得税季度预缴

**路径**：会计 → 报告 → 中国税务申报 → **企业所得税预缴**

**操作步骤**：

1. 选择公司和季度
2. 确认「小型微利企业」勾选状态是否正确
3. 填写「前期已缴税额」（本年前几季度已缴纳的金额）
4. 点击「**计算**」

| 字段 | 说明 |
|------|------|
| 累计收入总额 | 年初至本季末，销售类科目贷方合计 |
| 累计成本费用总额 | 年初至本季末，费用类科目借方合计 |
| 累计利润总额 | 收入 - 成本费用 |
| 适用税率 | 小微≤100万：5%；100~300万：10%；>300万或非小微：25% |
| 本期应预缴税额 | 累计应纳税额 - 前期已缴税额 |

5. 将数据填入电子税务局「企业所得税月（季）度预缴申报表（A类）」
6. 纳税凭证：

```
借：所得税费用（5801）              本期预缴税额
  贷：应交税费—应交企业所得税（2221）  本期预缴税额

借：应交税费—应交企业所得税          实际缴纳额
  贷：银行存款                       实际缴纳额
```

### 4.3 申报工作流总览

```
季末最后一天
  ↓
① Odoo：增值税申报汇总 → 计算 → 记录数字
② Odoo：利润表 → 计算 → 打印 PDF（留档）
③ Odoo：资产负债表 → 计算 → 打印 PDF（留档）
④ Odoo：企所税预缴 → 计算 → 记录预缴金额
  ↓
季后15日内
  ↓
⑤ 电子税务局：填写增值税申报表 → 提交 → 缴纳
⑥ 电子税务局：填写企所税预缴表 → 提交 → 缴纳
  ↓
⑦ Odoo：录入纳税付款凭证（借：应交税费；贷：银行存款）
```

## 五、中国会计准则概要

### 5.1 两套准则对比

| 维度 | 小企业会计准则 | 企业会计准则 |
|------|--------------|------------|
| 适用对象 | 不对外发行证券、规模较小的企业 | 上市公司及大型企业 |
| 科目体系 | 简化（约113个） | 完整（155个+） |
| 本系统对应 | `template_cn` | `template_cn_large_bis` |
| 固定资产折旧 | 直线法为主，方法简单 | 多种折旧方法，需披露 |
| 合并报表 | 不强制 | 母公司必须编制 |
| 年报审计 | 非强制（贷款/政府要求除外） | 上市公司强制 |
| 主要依据 | 财政部 2013 年《小企业会计准则》 | 财政部 2006 年《企业会计准则》及修订 |

## 六、主要税种速查

### 6.1 增值税

| 项目 | 小规模纳税人 | 一般纳税人 |
|------|------------|---------|
| 税率 | 3%（正常）/ 1%（优惠） | 13% / 9% / 6% |
| 进项抵扣 | **不可抵扣** | **可抵扣**（需专用发票） |
| 计税方式 | 简易计税：不含税额 = 含税价 ÷ 1.03 × 3% | 一般计税：销项税 - 进项税 |
| 申报周期 | 按**季**申报 | 按**月**申报 |
| 发票类型 | 增值税普通发票 | 可开增值税专用发票 |

**附加税（随增值税同申报缴纳）**：

| 税种 | 计税基数 | 税率 |
|------|---------|------|
| 城市维护建设税 | 实际缴纳增值税额 | 7%（城区）/ 5%（县镇）/ 1%（其他） |
| 教育费附加 | 同上 | 3% |
| 地方教育附加 | 同上 | 2% |

### 6.2 企业所得税

| 项目 | 内容 |
|------|------|
| 标准税率 | **25%** |
| 小型微利企业优惠 | 年利润 ≤ 100万：**5%**；100万~300万：**10%** |
| 高新技术企业 | **15%**（需认证） |
| 申报方式 | 季度**预缴** + 次年5月31日**汇算清缴** |
| 小微判定条件 | 年利润 ≤ 300万 + 从业人数 ≤ 300人 + 资产总额 ≤ 5000万（工业）或 300人/3000万（其他） |

### 6.3 个人所得税（代扣代缴）

| 情形 | 税率 | 说明 |
|------|------|------|
| 工资薪金 | 3%~45%（七级超额累进） | 月扣除5000元起征点后适用 |

## 九、财税基础知识

### 9.1 会计基本方程式

```
资产 = 负债 + 所有者权益

利润 = 收入 - 成本费用 - 税金及附加

现金净流量 = 经营活动净流量 + 投资活动净流量 + 筹资活动净流量
```

### 9.2 借贷记账法规则

```
资产类账户：    借方增加，贷方减少
负债/权益类账户：贷方增加，借方减少
收入类账户：    贷方增加，借方结转（期末清零）
费用/成本类账户：借方增加，贷方结转（期末清零）
```

**口诀**：有借必有贷，借贷必相等。

### 9.3 典型会计分录示例

**销售商品（小规模纳税人，含税价 103 元）**：
```
借：应收账款              103.00
  贷：主营业务收入（5001）  100.00
       应交税费—应交增值税    3.00
```

**采购原材料（一般纳税人，含税价 113 元，进项可抵扣）**：
```
借：原材料（1403）                        100.00
    应交税费—应交增值税（进项税额）（2221）  13.00
  贷：应付账款（2202）                    113.00
```

**计提固定资产折旧（月折旧 833.33 元）**：
```
借：管理费用—折旧费（5602）    833.33
  贷：累计折旧（1602）          833.33
```

**缴纳增值税及附加税**：
```
借：应交税费—应交增值税       3,000.00
    应交税费—城市维护建设税      210.00
    应交税费—教育费附加          90.00
    应交税费—地方教育附加        60.00
  贷：银行存款（1002）         3,360.00
```

**缴纳企业所得税季度预缴**：
```
借：所得税费用（5801）                    5,000.00
  贷：应交税费—应交企业所得税（2221）      5,000.00

借：应交税费—应交企业所得税              5,000.00
  贷：银行存款（1002）                   5,000.00
```

### 9.4 纳税人身份选择建议

| 考量因素 | 建议 |
|---------|------|
| 年销售额 ≤ 500万，客户为个人消费者 | 保持**小规模**，税负更低 |
| 上游客户需要专票抵扣（B2B 制造/贸易） | 认定**一般纳税人**，维持抵扣链 |
| 原材料/人工成本占比高（增值率低） | **一般纳税人**更划算（可抵扣大量进项） |
| 已超年销售额 500 万标准 | **必须**申请认定一般纳税人 |

### 9.5 小型微利企业所得税优惠（现行政策）

年应纳税所得额 **≤ 100 万元**：
- 应纳税额 = 利润 × 25% × 20% = 利润 × **5%**

年应纳税所得额 **> 100 万元且 ≤ 300 万元**：
- 应纳税额 = 100 万 × 5% + (利润 - 100 万) × 25% × 40%
- 即 100 万以内 5%，超出部分 **10%**

> 以上为优惠税率，需同时满足从业人数和资产规模条件。政策每年可能调整，以财政部/税务总局当年公告为准。

### 9.6 年度汇算清缴要点

企业所得税年度汇算清缴截止 **5月31日**，主要步骤：

1. **核对账务**：确保全年所有凭证已入账，无悬空科目，期末科目余额合理
2. **生成报表**：在 Odoo 生成全年资产负债表和利润表，打印 PDF 留存
3. **纳税调整**（在电子税务局申报表中填写）：
   - **调增**：业务招待费超标部分、罚款滞纳金、赞助费等
   - **调减**：研发费用加计扣除、政策性减免等
4. **计算年度应纳税额** = （利润总额 ± 纳税调整额）× 适用税率
5. **补缴或退税**：年度税额 - 全年已预缴合计 = 补缴（退税）金额

**常见纳税调增项目**：

| 项目 | 扣除限额 |
|------|---------|
| 业务招待费 | 实际发生额 × 60%，且 ≤ 当年销售收入 × 5‰ |
| 广告宣传费 | ≤ 销售收入 × 15%（一般行业） |
| 职工福利费 | ≤ 工资薪金总额 × 14% |
| 工会经费 | ≤ 工资薪金总额 × 2% |
| 职工教育经费 | ≤ 工资薪金总额 × 8% |
| 罚款、税务滞纳金 | **全额调增**，不可扣除 |

## 十、常见问题

**Q：报表打印出来后能直接交给税务局吗？**
A：需区分报表类型。本系统生成的**资产负债表/利润表**格式符合《小企业会计准则》，可作为汇算清缴的财务报表附件提交。但**增值税申报表、企所税预缴表**等申报表仍需在**电子税务局**填写提交，不能用 Odoo 打印件代替。

**Q：资产总计和负债+权益总计不相等怎么办？**
A：说明账务有问题。常见原因：① 有已过账的凭证借贷不平衡（Odoo 不允许）；② 某些科目归属不正确（如费用类科目被归入资产类）；③ 存在跨公司凭证未正确处理。建议先用「试算平衡表」核查各科目余额，找出异常科目。

**Q：小规模纳税人收到增值税专用发票，能抵扣吗？**
A：不能。小规模纳税人采用简易计税，进项税不可抵扣，收到专票后将**含税金额全额计入成本费用**。

**Q：公司当季无收入，还需要申报吗？**
A：需要。即使零收入也必须按时**零申报**，否则将被列为非正常户，影响发票领用和信用等级。

**Q：多公司情况下，申报数据会不会混淆？**
A：不会。Odoo 右上角切换公司后，所有报表和向导**自动只显示当前公司数据**。建议每家公司的财务人员仅登录本公司账号，并通过权限组严格管控访问范围。

**Q：利润表显示的数字与增值税申报汇总的销售额对不上？**
A：两者口径不同：① 利润表的营业收入按**会计科目（5001/5051）记账日期**汇总；② 增值税申报按**发票日期**且只计已过账的销售发票。如果有未开票收入或收款与开票不同期，数字会有差异，属正常现象。

**Q：企业所得税预缴的利润数字需要做纳税调整吗？**
A：季度**预缴**阶段通常按账面利润预缴，不强制要求纳税调整。重大调整事项（如大额招待费超标、大额研发加计扣除）建议在**年度汇算清缴**时统一处理，预缴阶段填入申报表的实际利润即可。

**Q：年末汇算清缴要补缴大量税款，如何规避滞纳金？**
A：季度预缴按实际利润足额预缴，避免年末大额补缴。滞纳金为每日**万分之五**，对补缴金额较大的企业影响显著。汇算时合理利用研发费用加计扣除、小微优惠等政策，可合法降低应纳税额。


 ##知识点
- **会计**‌：聚焦于‌**记录过去**‌，核心是‌**核算与监督**‌。它通过确认、计量、记录和报告企业经济活动，提供真实、合规的财务信息（如资产负债表、利润表等），主要依据《企业会计准则》和税法，强调“有据可查”‌34。
- ‌**财务管理**‌：着眼于‌**规划未来**‌，核心是‌**决策与控制**‌。它基于会计提供的信息，进行筹资、投资、营运资金及利润分配等决策，目标是实现企业价值最大化，强调“用钱生钱”‌35。
- 会计是“账房先生”，负责‌**记账、报账、合规**‌；
- 财务管理是“战略参谋”，负责‌**分析、预测、优化资金使用**
